
# Web Foundations Todo List

Simple Todo app with CRUD operations

### First Steps
```
$ python3 -m venv <name_of_virtualenv>
$ . <name_of_virtualenv>/bin/activate
$ pip install -r requirements.txt
```
### Run
```
$ python3 app.py
```
